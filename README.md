# todo-app
### This is a minimalist todo app using React / AWS / Tailwind.css / Framer Motion

App is running at https://master.d3fb1wjw2da8y7.amplifyapp.com/

Images


###### Requirements:
- NPM (Node Package Manager)
- React.js
- Tailwind.css
- Framer Motion
- AWS Amplify CLI

###### Usage:
Run `npm start` to start app on port 3030
